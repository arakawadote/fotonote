# Google Playストア掲載文案：FotoNote

最終更新日：2026-05-12

## 公式入力制限メモ

Google Playのメインストア掲載情報では、以下の文字数制限があります。

- アプリ名：30文字まで
- 短い説明：80文字まで
- 詳細説明：4000文字まで

参照：Google Play Console Help「Create and set up your app」
https://support.google.com/googleplay/android-developer/answer/9859152

## 詳細説明

```txt
FotoNoteは、写真に含まれるEXIF情報を読み取り、カメラ情報付きのシンプルなフレーム画像を作成できるAndroidアプリです。

写真を選ぶだけで、カメラメーカー名、機種名、焦点距離、F値、シャッタースピード、ISOなどの撮影情報を整えて表示できます。Classic White、Soft Gray、Noirの3種類からテンプレートを選び、作成した画像を端末のギャラリーに保存できます。

主な機能
- 端末内の写真を選択
- 写真に含まれるEXIF情報を読み取り
- カメラ情報付きのフレーム画像を作成
- 3種類のテンプレートを選択
- 作成した画像をギャラリーに保存
- 縦写真・横写真の向きに対応
- EXIF情報がない画像でも利用可能

プライバシーへの配慮
- 写真とEXIF情報は端末内で処理します
- 選択した写真を外部サーバーへ送信しません
- GPS情報は画面や書き出し画像に表示しません

FotoNoteは、写真と撮影情報をシンプルに残したい方、SNSやアルバム用にカメラ情報付き画像を作りたい方に向けたアプリです。
```
